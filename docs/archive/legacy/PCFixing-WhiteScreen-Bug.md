# Win11 白屏闪烁故障原因及解决方法

## 故障原因

一句话概括：KB5022913 补丁和 startallback 插件有冲突引起的

参考链接：

- https://www.chiphell.com/thread-2500922-1-1.html
- https://www.chiphell.com/thread-2500697-1-1.html

## 解决方案

强制重启三次电脑，自动进入安全模式后，卸载 KB5022913 补丁或 startallback 插件即可

也可进入 PE 进行卸载。
