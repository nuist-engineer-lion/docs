## 记忆介质

用记忆介质这个名字是为了体现出内存和硬盘共性的地方，就是记忆（memory）。

内存（RAM）和硬盘（ROM）是许多小白都会混淆的地方之一。内存在电脑中是对应的内存条，而很多人说电脑没储存空间了则是指的硬盘。内存是电脑在运行过程中间正在运行的程序所存放的高速储存空间，其特点是速度快而一但断电其内部的数据就会消失，比如用台式机写Word文档突然断电的话内容就会消失，这就是因为Word在编辑中数据是储存在内存里的，所以最好要常常保存避免数据损失；硬盘则是电脑的储存空间，通常对于普通家用电脑来说，硬盘的容量就体现在“此电脑”中“C盘”“D盘”等的大小上

### 内存

用于短时间的快速存取，相当于人的大脑的记忆功能。内存越大，能够同时运行的程序就越多。

### 硬盘

用于长时间的存储，类似于把事情记在本子上。硬盘存储空间越大，能存的程序就越多。

#### 固态硬盘

利用芯片存储信息，不通电的情况下保存时间会比机械短一些，但也会有几年之久。现在的笔记本一般只留了给固态硬盘的位置。

#### 机械硬盘

利用可磁化的盘片存储信息，因为是机械结构，所以更容易损坏，使用的时候请不要震动他。相比固态硬盘会更便宜一些，但速度更慢。

## 散热模块

1. 性能/散热

   1. 首先在日常电脑的使用中，笔记本是最好要一直插着电的，在充电状态下笔记本才能充分释放性能。很多人担心长期充电损害电池，实际充电器给笔记本供电是直通主板，并不影响电池，所以请不要一充满电池就拔掉充电器，这反而才对电池不好；
   2. 其次散热是笔记本性能释放的重要保证，想促进散热有以下办法：
   
      - 保持机身进风口和出风口无异物挡住（比如不放在被子上面使用）；
      - 可以将机身用瓶盖、支架或者烧烤架垫起来，使机身散热模组可以与外界气体充分流动交换（注：如果电脑使用了液金散热材料则本条不适用，请参考下面液金使用攻略）；
      - 定期清灰更换硅脂，尽量保持至少一年一次的频率，如果电脑长期处于多灰环境则更要定期检查更换以免电脑卡顿损毁（如果电脑卡顿了请先检查一下电脑散热口）。

1. 液金/硅脂：

   1. 液金和硅脂都是用来填充电脑内部散热模组与cpu核心/gpu核心之间细微缝隙的导热材料。在笔记本上由于近年来发热的逐渐增加，常规硅脂的导热能力已经成为了电脑散热的瓶颈之一。因此，新型液金材料成为这两年一线游戏本的首选，液金的导热性能优于硅脂，在常温下液金呈固态，在温度稍高的情况下液金呈液态；
   2. 液金使用攻略：
   
      - 上面也提到了液金在温度较高的情况下会呈液态，因此在电脑使用过程中内部温度会让液金一直处于可流动的液态，所以在使用中液金笔记本要平放，不然液金会偏向一侧形成液金偏移，导致cpu/gpu高温，所以如果添加上文所提到的支架的话，只能使用平放的烧烤架；
      - 在电脑使用结束后最好不要立刻将电脑装包移动，此时内部液金仍然是液态，会液金偏移；
   3. 液金鉴别：想知道自己的电脑是不是液金的话，可以询问店家或者在网上搜索对应年份的对应机型是否采用了液金。
