# TypeScript

如果你想快速进入到我们的开发工程中，你不必过多深入了解`TypeScript`，只需要大概了解类型标注即可。

如果你有别的语言的经验，你也可以先阅读已有的代码，相信你很快会对ts的使用有基础的了解，然后进入的项目的开发中。
