## 2.2 颜值/纯性能

颜值：光污染黑海/白海   无光黑海/白海

 PS：颜值默认水冷，有漏液以及使用寿命，需要进行说明是否使用水冷/风冷（风冷颜值会下降）

纯性能：纯性价比/无丐

相同性能下：

纯性价比+500=无丐

无丐+100=无光黑海+150=光污染黑海

无丐+300=无光白海+200=光污染白海

PS：价格经供参考，具体要看我们的进货价
